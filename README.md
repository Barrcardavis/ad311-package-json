# AD311 – package.json Assignment

This project explores the structure and purpose of the `package.json` file in a Node.js application.  
It includes dependencies, devDependencies, custom scripts, engines, and other required fields.

## How to Run the Project

Install packages:
npm install

Run the main program:
npm start

Run the test cases:
npm test

## Files Included

- `package.json` – fully completed with all required fields  
- `index.js` – simple entry point  
- `test.js` – includes 3 normal and 3 edge-case tests  
- `package-lock.json` – automatically generated by npm  

## Purpose

This assignment demonstrates:
- Understanding of package.json fields  
- Managing dependencies with npm  
- Creating custom scripts  
- Running test cases  
- Publishing to a public GitHub repository
